# 贪吃蛇游戏 🐍

一个使用原生JavaScript实现的网页版贪吃蛇游戏，具有精美的视觉效果和多种地图选择。

## 功能特点

- 🎨 精美的渐变背景和卡通风格的蛇形象
- 🗺️ 三种地图大小可选（小、中、大）
- 🎯 彩色食物带光晕效果
- 📊 分数记录和最高分保存
- ⌨️ 流畅的键盘控制
- 📱 响应式设计

## 如何运行

1. 克隆或下载此仓库
2. 直接在浏览器中打开 `index.html` 文件
3. 使用方向键控制蛇的移动
4. 按空格键或任意方向键开始游戏

## 游戏规则

- 使用方向键（↑ ↓ ← →）控制蛇的移动方向
- 吃到食物可以增加分数和蛇的长度
- 撞到墙壁或自己的身体游戏结束
- 可以在游戏开始前选择不同的地图大小

## 技术栈

- HTML5 Canvas
- CSS3
- 原生JavaScript (ES6+)
- LocalStorage (用于保存最高分)
